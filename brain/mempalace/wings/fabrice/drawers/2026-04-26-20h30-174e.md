## OPTION A — Tester le scan proactif (Couche B, sans install)

N'importe quel store Shopify public suffit. Stores connus et accessibles :

| Store | URL à entrer dans StoreMD |
|---|---|
| Gymshark | `gymshark.com` |
| Allbirds | `allbirds.com` |
| Kylie Cosmetics | `kyliecosmetics.com` |
| Bulletproof | `bulletproof.com` |

Tu n'as besoin que de l'URL publique — StoreMD crawle la vitrine, analyse les scripts, détecte les apps via les balises injectées. Zéro OAuth pour cette partie.

---

## OPTION B — Tester le flow complet (OAuth + install + billing)

Il te faut un **Shopify Partner development store** — gratuit, complet, parfait pour les tests.

**Étapes exactes (5 minutes) :**

1. Aller sur **partners.shopify.com** → créer un compte gratuit
2. Dans le dashboard Partners → **Stores** → **Add store**
3. Choisir **Development store**
4. Donner un nom (ex: `f2-test-store`) → Create store
5. Ton URL sera `f2-test-store.myshopify.com`

Avantage : tu peux créer des faux produits, simuler du billing, tester les ghost charges en installant/désinstallant des apps gratuites pour créer des `recurring_application_charges` orphelines.

---

**Mon conseil** : commence par Option A avec `gymshark.com` pour valider que les scanners retournent des vraies données. Ensuite Partner store pour tester le flow OAuth + billing Stripe.
