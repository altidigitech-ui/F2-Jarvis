## Install sans App Store — comment ça marche

Tout app créée dans le **Shopify Partner Dashboard** génère automatiquement un lien d'installation direct, même si elle n'est jamais soumise à la review App Store.

L'URL ressemble à ça :

```
https://{store}.myshopify.com/admin/oauth/authorize
  ?client_id={TON_API_KEY}
  &scope={SCOPES}
  &redirect_uri={TON_REDIRECT_URI}
  &state={NONCE}
```

Le merchant clique → accepte les permissions OAuth → installée. Exactement comme une app publique, sauf que l'URL vient de toi directement.

## Ce que ça change pour StoreMD

Tu as **deux modes d'installation** disponibles maintenant :

| Mode | Quand l'utiliser |
|---|---|
| **Lien direct** (maintenant) | Beta testers, cold DM, Couche B — tu envoies le lien, ils installent en 30s |
| **App Store** (plus tard) | Distribution organique, featured placement, credibilité |

Pour la **Couche B S7**, le flow proactif devient :
1. Tu scannes le store sans install (preview scan)
2. Tu envoies le rapport dans un DM
3. Tu joins le lien d'install direct en fin de message : "if you want the full scan with fixes, install here → [lien]"
4. Ils installent, OAuth passe, les write scopes sont actifs

## Ce qu'il faut vérifier

Ton app est dans le Partner Dashboard avec l'API Key configurée ? Si oui, le lien d'install existe déjà. Il suffit de le construire avec tes valeurs `SHOPIFY_API_KEY`, `SHOPIFY_SCOPES` (depuis `config.py`), et ton `REDIRECT_URI` (depuis les env vars prod).
