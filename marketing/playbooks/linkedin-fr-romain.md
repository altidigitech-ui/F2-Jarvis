# Playbook — LinkedIn FR Romain

## Objectif

Construire une audience FR founders / marketeurs qui associe F2 à :
- Expertise SaaS indie
- Build-in-public authentique
- Learning techniques et business

## Cadence

- **2 à 3 posts / semaine** idéal
- **Min 1 post / semaine** (jamais de silence > 10 jours)
- **Pas de "post pour poster"** — quality > quantity

## Types de posts

### 1. Launch announcement (F2 SaaS)
Voir template dans `launch-saas.md`.

### 2. Learning technique
"On a fait X. Voilà ce qu'on a appris."
Format : observation + mécanisme + implication pratique.

### 3. Observation écosystème
"J'ai vu X chez 100 SaaS FR. Voilà le pattern."
Format : constat + 2-3 exemples + reco.

### 4. Métrique F2 brute
"MRR F2 avril : 420 €. 2 fondateurs. Voilà ce qui a marché."
Format : chiffre + context + 2-3 causes identifiées.

### 5. Post-mortem honnête
"On a pivoté Leak Detector. Pourquoi."
Format : décision + raisons + conséquences + leçons.

### 6. Question ouverte à l'audience
"Comment vous gérez X ?" (rare, 1/10 posts max, sinon feels like engagement bait)

## Horaires

- **Mardi, mercredi, jeudi** : meilleur engagement LinkedIn FR
- **8h-10h CEST** : peak matinal
- **12h-13h CEST** : peak lunch
- **Éviter** : lundi matin (inbox rush), vendredi soir (tout le monde décroche)

## Engagement

### Les 2 premières heures
- Répondre à TOUS les commentaires
- Répondre avec substance (pas "Merci !")
- Liker les commentaires qualifiés
- Ne PAS retweeter/partager son propre post dans la foulée

### Les 24 premières heures
- Répondre dans les 4h
- Si débat intéressant, l'entretenir (pas fuir)
- Screenshoter les meilleurs échanges pour futurs posts

### Après 48h
- Le post est "mort" algorithmiquement
- Répondre quand même aux commentaires tardifs (classe)

## Règles strictes

- **Pas de posts auto-générés** par AI sans review humaine
- **Pas de copier-coller** d'un post Twitter vers LinkedIn
- **Pas de threads** (LinkedIn ≠ Twitter)
- **Pas de hashtag generic** (#entrepreneur #motivation)
- **Pas de cover Canva "Entrepreneur Journey"**
- **TOILE rule** : zéro mention Altistone

## Métriques à tracker

Dans `marketing/analytics/linkedin-stats.csv` :
- Impressions
- Reactions (likes + autres)
- Comments
- Reposts
- Profile views
- Click-through si lien

À reporter dans metrics-weekly, pas obsession daily.

## Style Romain checklist

Avant publication, vérifier :
- [ ] Longueur 500-1500 chars
- [ ] 0 à 2 emojis
- [ ] 0 à 3 hashtags
- [ ] Aéré (retours à la ligne fréquents)
- [ ] Pas de "Voici 5 hacks"
- [ ] Pas de "La vérité que personne ne dit"
- [ ] Pas de fake stats
- [ ] Screenshots réels si présents
- [ ] Date explicite si chiffre
- [ ] TOILE rule respectée
