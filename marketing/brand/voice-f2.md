# F2 — Brand Voice

## Ce que F2 est

Studio SaaS indie, 2 fondateurs (Fabrice + Romain), build-in-public, cadence 1 SaaS/mois.

## Ce que F2 n'est pas

- Pas une startup qui lève des fonds
- Pas une agence qui build pour d'autres
- Pas un media / newsletter / course
- Pas "the future of work" / "revolutionary AI"

## Tone global

- **Authentique** : on partage les chiffres réels, les échecs, les pivots
- **Technique mais accessible** : on écrit pour des founders/devs, pas pour des VCs
- **Direct** : phrases courtes, pas de "we are thrilled to announce"
- **Observateur** : on commente l'écosystème avec recul
- **Parfois sec, jamais méchant** : on peut être tranchant, pas insultant

## Principes non-négociables

### 1. Zéro fake stats
Aucun "10,000+ users", "trusted by 500 companies", "3x faster" sans benchmark public reproductible.

### 2. Zéro fake testimonials
Pas de "Sarah, CEO" fictifs. Pas de quotes inventées.

### 3. Screenshots réels
Pas de mockups trompeurs. Les screenshots montrent le vrai produit.

### 4. Dates explicites
"En avril 2026" plutôt que "récemment". "MRR avril 2026 : 420 €" plutôt que "growing fast".

### 5. Build-in-public sincère
Partager les MRR réels, les features cassées, les bugs, les pivots.

### 6. TOILE rule
Altistone n'existe pas publiquement.

### 7. Pas de hype AI
F2 utilise l'AI, mais on ne vend pas "AI-powered X". On vend ce que le produit fait concrètement.
